HTML5

HTML5 es la última versión del lenguaje de marcado estándar para crear y diseñar páginas web. Con mejoras significativas en la estructura ofrece etiquetas semánticas para la organización del contenido, así como soporte integrado para audio, video y gráficos vectoriales.

CSS3

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para controlar la presentación de las páginas web. Es responsable del diseño responsivo y con su última "versión", CSS3 permite además crear efectos de transición y animación increíbles para lograr diseños modernos y atractivos.

JavaScript

JavaScript es un lenguaje de programación interpretado ampliamente utilizado en el desarrollo web. Es responsable de agregar interactividad a las páginas web haciendo posible la interacción con el usuario tan necesaria para crear experiencias web ricas y sofisticadas.
